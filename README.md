## 介绍

针对大多数学校机房中安装的Cisco PT 6.2sv和官网最新的8.2.1版本制作的汉化包
以及8.2.1版本的免登录补丁

- languages: Cisco PT 的语言包文件夹
- help: Cisco PT 的帮助文件夹


## 下载
